Linha de Estudos HTML e CSS

Pratique os 10 desafios e veja se acertou!

HTML - Fundamentos

Crie uma página com:
  • Um <h1>
  • Um <p> com seu nome e idade
  • Uma linha horizontal <hr>
  • Uma quebra de linha <br>

HTML - Listas e Links

Crie uma lista <ul> com 5 itens <li>. Cada item deve ser um link.

HTML - Imagens e Mídia

Crie uma página com pelo menos 3 imagens e um vídeo.

HTML - Formulários

Crie um formulário de cadastro com:
  • Nome, e-mail, senha
  • Campo de seleção com 3 opções
  • Botão enviar

HTML - Semântica

Crie um site com:
  • Cabeçalho com título e menu
  • Uma seção com um artigo (parágrafo)
  • Rodapé com informações de contato

CSS - Fundamentos

Pegue uma página HTML e:
  • Mude a cor do fundo
  • Mude a cor do texto
  • Escolha fontes diferentes para títulos e parágrafos

CSS - Box Model

Crie 3 divs com tamanhos, margens, padding e bordas diferentes.

CSS - Posicionamento

Crie uma página com:
  • Menu fixo no topo
  • Imagem no canto inferior direito
  • Parágrafo que se ajusta ao redor da imagem

CSS - Flexbox

Crie 3 cartões lado a lado usando flexbox, centralizados vertical e horizontalmente.

CSS - Animações e Transições

Crie um botão que muda de cor e aumenta de tamanho quando o mouse passa por cima.
