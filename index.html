Linha de Estudos HTML e CSS

Pratique todos os desafios de HTML e CSS neste site

HTML - Fundamentos

Desafio: Crie uma página com:
  • Um título principal <h1>
  • Um parágrafo <p> com seu nome e idade
  • Uma linha horizontal <hr>
  • Uma quebra de linha <br>

HTML - Listas e Links

Desafio: Crie uma lista de 5 livros ou filmes favoritos. Cada item deve ser um link.

HTML - Imagens e Mídia

Desafio: Crie uma página de "meus hobbies" com pelo menos 3 imagens e um vídeo.

HTML - Formulários

Desafio: Crie um formulário de cadastro com:
  • Nome, e-mail e senha
  • Campo de seleção "Curso favorito" com 3 opções
  • Botão "Enviar"

HTML - Estrutura Semântica

Desafio: Crie um site simples com:
  • Cabeçalho com título e menu
  • Uma seção com um artigo (parágrafo)
  • Rodapé com informações de contato

CSS - Fundamentos

Desafio: Pegue sua página HTML de teste e:
  • Mude a cor do fundo
  • Mude a cor do texto
  • Escolha uma fonte diferente para títulos e parágrafos

CSS - Box Model

Desafio: Crie 3 caixas (divs) com tamanhos, margens, padding e bordas diferentes.

CSS - Posicionamento e Display

Desafio: Crie uma página com:
  • Menu fixo no topo
  • Imagem posicionada no canto inferior direito
  • Parágrafo que se ajusta ao redor da imagem

CSS - Flexbox

Desafio: Crie um layout com 3 cartões lado a lado usando flexbox, centralizados vertical e horizontalmente.

CSS - Animações e Transições

Desafio: Crie um botão que muda de cor e aumenta de tamanho quando o mouse passa por cima.
